**3. Project Title**

**4. Dataset Information**

* **Dataset Title:** Online Retail Sales Dataset
* **Source Website / URL:** Kaggle (Online Retail Dataset)
* **Description:** The dataset contains transactional data of an online retail store, including invoices, products, quantities, prices, and customer information.
* **Why this dataset was selected:** It is suitable for data cleaning, aggregation, customer behavior analysis, and basic machine learning exploration.
* **Dataset Size:** Approximately 500,000 rows and 8 columns.

**5. Project Objectives**

* Analyze customer purchasing behavior using sales data.
* Identify top-selling products and high-value customers.
* Discover trends and patterns that can support business decision-making.

---

**6. Data Preparation (Using Pandas)**
The dataset will be prepared using Python Pandas library.

* Loading the dataset using `read_csv()`
* Handling missing values and removing duplicates
* Converting data types and creating new features such as total sales and order frequency

**7. Data Analysis Tasks**

* Filtering sales data by country and date
* Grouping and aggregating data to calculate total revenue
* Creating pivot tables for product and customer analysis
* Answering the business questions defined in the objectives

**8. Key Findings and Insights**

* Clear seasonal and sales trends observed in the data
